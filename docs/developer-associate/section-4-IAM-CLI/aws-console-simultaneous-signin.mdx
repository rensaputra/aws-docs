# 4.3 AWS Console Simultaneous Sign-In

Maarek shows how to enable and use the multi-session feature

## Key takeaways

![Multisession-support](./img/multisession-support.png)

- **Multi-session support**: Users can sign into multiple AWS accounts or roles within the same browser, facilitating easier account management.
- **Session management**: To enable multi-session support, user can click the "Turn on multi-session support" button in the AWS Management Console account settings.
- **Session switching**: Once multi-session support is enabled, users can add new sessions by clicking the "Add new session" button, allowing them to sign into different accounts or roles without getting kicked out of the current session.
